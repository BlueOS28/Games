# Skin Tone Texture Palette

A collection of 8 photorealistic skin tone textures for GTA character models.

## Included Tones

1. **deep-ebony.png** - Deep dark brown tone
2. **dark-umber.png** - Rich dark brown
3. **medium-mahogany.png** - Warm medium tone
4. **tan-sand.png** - Light warm tone
5. **fair-honey.png** - Fair warm tone
6. **pale-ivory.png** - Very fair neutral tone
7. **porcelain.png** - Very light pale tone
8. **peachy-blush.png** - Fair peachy tone

## Specifications

- **Format**: PNG with alpha channel
- **Detail**: Micro skin pore details for photorealistic appearance
- **Lighting**: Studio key + fill lighting
- **Finish**: Matte diffuse reflection
- **Grid Layout**: 2x4 arrangement for palette sheet display
